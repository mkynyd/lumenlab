# LumenLab Skills

## 论文学术 (academic)

- **paper-reader** -- 论文速读：三层深度阅读（裸读/引导/精读）+ 多论文对比，支持 DOCX 导出。适用场景：读 arXiv 论文、精读 PDF、多篇论文对比分析。
- **paper-writer** -- 论文写作助手：读取资料、联网检索文献（arXiv/Wikipedia/OpenReview）、整理引用、保存草稿。适用场景：撰写论文初稿、修改章节、整理参考文献。

## 考试复习 (exam)

- **exam-coach** -- 复习教练：五步迭代循环（评估-计划-材料-实战-弱项复盘），自动生成闪卡和自测题。适用场景：期末备考、错题整理、制定复习计划。
- **exam-extract** -- 考点分析：大纲驱动的 8 字段考点抽取，支持 triage 模式按权重排序。适用场景：考前划重点、从课件 PPT 提取考点、时间紧张时的优先级复习。

## 编程技术 (coding)

- **code-reader** -- 代码阅读：抓取 GitHub 公开仓库，分析架构、核心模块和调用关系，生成 Markdown 报告。适用场景：理解陌生开源项目、代码 review 前的架构概览。

## 通识学习 (learning)

- **socratic-tutor** -- 苏格拉底导师：三阶段引导式提问（引导问题-关键提示-下一步建议），不直接给答案。适用场景：数学/物理/编程/经管/文史/论文等全学科的启发式辅导。
