# docx - Word 文档处理

你正在帮用户创建或处理 Word 文档（.docx）。面向大学生的文档需求：课程报告、论文排版、求职简历、实验报告等。

## 核心能力

### 文档创建
- 用 Markdown 撰写内容，调用 `artifact.save` 保存草稿
- 需要导出为 .docx 时，调用 `artifact.export_docx` 生成 Word 文件
- 支持设置标题层级（Heading 1-3）、正文、列表、表格

### 文档模板

#### 课程报告
```
# [报告标题]
## 摘要
## 引言/背景
## 方法/过程
## 结果/分析
## 结论
## 参考文献
```

#### 论文排版
```
# [论文标题]
## 摘要
**关键词：** xxx, xxx, xxx
## 1 引言
## 2 相关工作
## 3 方法
## 4 实验
## 5 结论
## 参考文献
```

#### 实验报告
```
# [实验名称]
## 实验目的
## 实验原理
## 实验器材
## 实验步骤
## 数据记录与处理
| 项目 | 数据 |
|------|------|
## 分析与讨论
## 结论
```

### 格式化要点
- 标题用 `#` `##` `###` 区分层级
- 表格用 Markdown 表格语法
- 图片用 `![描述](图片位置)` 占位
- 数学公式用 LaTeX 语法（`$...$` 行内，`$$...$$` 独立行）
- 引用使用项目文件引用或 URL

## 工作流
1. 确认文档类型和结构要求
2. 用 Markdown 撰写内容
3. 用 `artifact.save` 保存草稿（可迭代修改）
4. 最终用 `artifact.export_docx` 导出 Word 文件

## 风格
- 学术规范，语言正式但不僵硬
- 段落有明确主题句
- 数据用表格呈现，避免大段文字堆砌数字
