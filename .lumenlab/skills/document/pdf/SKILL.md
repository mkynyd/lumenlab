# pdf - PDF 文档处理

你正在帮用户处理 PDF 文件。面向大学生的 PDF 需求：读教材/论文 PDF、提取笔记、合并课件、处理扫描件。

## 核心能力

### PDF 阅读
- 用 `project_files.read` 读取已上传到项目的 PDF（已自动解析文本）
- 对于未解析的 PDF，先确认是否已在项目中
- 提取特定章节/页码的内容

### PDF 内容提取
- 提取文本：精确到章节和页码（如果 PDF 已解析）
- 提取表格数据：识别 PDF 中的表格并转为 Markdown 表格或 CSV
- 提取图片/图表：标注图片位置和描述

### PDF 信息查看
- 查看页数、文件大小
- 查看是否包含图片/扫描件
- 判断 PDF 类型（文本型/扫描型）

### 注意事项
- 扫描件 PDF（图片型）无法直接提取文字，需告知用户"这是扫描件，无法直接提取文本"
- 受保护的 PDF 无法处理，需告知用户
- 大文件（>50MB）建议用户先在本地拆分

## 工作流
1. 确认 PDF 已在项目中（`project_files.list`）
2. 用 `project_files.read` 读取内容（指定 fileId）
3. 根据用户需求提取/分析/总结内容
4. 结果保存为 `artifact.save`（可选）

## 与其他 skill 配合
- 论文 PDF → 配合 `paper-reader` 进行深度阅读
- 课件 PDF → 配合 `exam-extract` 提取考点
- 教材 PDF → 配合 `exam-coach` 生成复习材料

## 风格
- 提取内容保持原文结构（章节、页码）
- 表格保留行列关系
- 中文为主，专业术语保留原文
