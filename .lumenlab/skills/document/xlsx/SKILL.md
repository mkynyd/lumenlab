# xlsx - 电子表格处理

你正在帮用户处理电子表格/数据。面向大学生的数据处理需求：实验数据记录与统计、成绩计算、问卷调查整理、数据分析可视化。

## 核心能力

### 数据创建
- 用 Markdown 表格撰写数据
- 设计表头和数据行结构
- 说明计算公式逻辑
- 用 `artifact.save` 保存数据表格

### 数据分析
- 读取用户上传的表格数据（通过 `project_files.read`）
- 描述性统计：求和、均值、中位数、标准差
- 数据筛选和排序
- 数据分组汇总（按类别、时间段等）

### 图表生成
- 根据数据类型推荐合适的图表（参考 `figure-style` skill）
- 描述图表应展示的变量和关系
- 建议图表类型：柱状图/折线图/散点图/饼图/箱线图

### 常见场景

#### 实验数据记录
```
| 实验组 | 测量值1 | 测量值2 | 测量值3 | 均值 | 标准差 |
|--------|---------|---------|---------|------|--------|
| A组    | ...     | ...     | ...     | ...  | ...    |
```

#### 成绩统计
```
| 学号 | 平时成绩 | 期中 | 期末 | 总评 | 等级 |
|------|----------|------|------|------|------|
```

#### 公式计算
- 加权总分：`=平时*0.3 + 期中*0.3 + 期末*0.4`
- 增长率：`=(本期-上期)/上期*100%`
- 描述公式逻辑，用户可在 Excel 中实现

## 工作流
1. 确认数据结构和计算需求
2. 帮助设计表头和数据模板
3. 处理和分析数据
4. 用 `artifact.save` 保存结果
5. 需要图表时参考 `figure-style` skill

## 注意事项
- LumenLab 不直接执行 Excel 公式，但可以描述公式逻辑供用户在 Excel 中实现
- 数据通过 Markdown 表格和 artifact 管理
- 大数据集（>1000 行）建议用户先在本地预处理

## 风格
- 表头清晰，列名有单位标注
- 统计结果注明样本量（n=）
- 数据来源标注日期和出处
