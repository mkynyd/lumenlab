## 产品定位

LumenLab 是一个**项目化**的 AI 学习工作台。区别于通用聊天工具，它把以下能力整合在同一个工作流中：

- 以项目为单位管理资料与上下文。
- 根据项目类型自动匹配提示词模板和快捷任务。
- 受控的 Agent 模式，AI 在执行重要操作前会请求你确认。
- 把优质回答保存为成果，支持导出 Markdown、DOCX、PDF。

## 目标用户

| 用户群 | 典型场景 |
|--------|----------|
| 计算机专业学生 | 实验报告、代码 review、复习提纲 |
| 通用大学生 | 课件解析、复习资料整理、笔记沉淀 |
| 课程助教 / 教师 | 作业批改、题目讲义、资料归档 |

桌面端是主要使用场景，移动端适合查看、轻量输入和状态确认。

## 核心特性

### 项目类型驱动

- 项目分为实验、复习、编程、通用四种类型。
- 不同类型会匹配不同的系统提示词和快捷任务，帮你更快进入状态。

### 项目化资料管理

- 上传图片、PDF、文本或代码文件，系统自动完成 OCR 或文本提取。
- 可在对话中勾选文件作为上下文，让 AI 基于真实资料回答。

### 受控 Agent 模式

- AI 可以在你授权后调用工具，例如搜索资料、保存成果、导出文档。
- 涉及写文件、导出、调用外部网页等操作前，系统会弹出审批卡片，由你决定允许或拒绝。

### 内置 Skills

- 提供论文写作、备考、代码阅读、论文阅读、考点提取、苏格拉底式辅导等 Skill。
- 每个 Skill 相当于一个场景模板，会自动收窄 AI 可使用的工具范围。

### 内置 Tools

- 项目文件管理：列出、读取、删除项目中的资料。
- 成果管理：保存、列出、导出 DOCX。
- 网页获取：在允许的站点范围内搜索和读取网页内容。
- 项目资料检索：基于项目资料进行关键词和语义搜索。
- 学术检索：搜索、读取、获取 arXiv 论文。
- 参考文献：添加、列出、关联和格式化参考文献。

### 多模型流式对话

- 支持 DeepSeek V4 Pro / Flash，复杂推理可开启深度推理模式。
- MiniMax M3 负责图片 OCR、PDF 文档解析等多模态任务。
- 流式输出，Markdown、公式、流程图、代码高亮实时渲染。

### PDF 文档转换

- 整篇 PDF 可转换为包含公式、表格、图片的 Markdown。
- 可下载包含 Markdown、图片目录、样式 PDF 与 DOCX 的完整 ZIP 包。
- 转换结果可直接保存为项目资料。

### 智能资料检索

- 根据你选中的文件和提问，自动选择最合适的检索策略。
- 小文件直接加载全文，大文件使用关键词或混合检索。
- 未配置语义检索时自动降级为关键词检索。

### Artifact 成果库

- 支持十余种成果标签，如实验报告、复习提纲、代码说明、计算过程等。
- 以 Markdown 为统一源，导出格式保持一致。
- 导出结果会被缓存，重复下载无需等待。

### 注册码与凭据管理

- 用户注册需要邮箱、密码与有效的注册码。
- 注册码由管理员或部署这个项目的开发者发放。
- 系统支持两种模式：
  - **中央凭证模式**：API Key 由管理员通过 `CredentialProfile` 集中配置，用户无需自行设置。
  - **自托管模式**：在 `src/lib/config.ts` 或环境变量中设置 `USER_API_KEYS_ENABLED=1`，用户可以自行保存 API Key（通过 `POST /api/user/api-keys` 或由管理员使用 `scripts/setup-api-key.ts` 设置）。

## 技术栈

LumenLab 是在线 Web 应用，无需安装，浏览器打开即可使用。

## 与同类产品的差异

- **项目化而非纯聊天**：每个项目独立配置系统提示词、默认模型与文件上下文，学习资料不会散在对话里。
- **可控 Agent 而非裸工具调用**：AI 调用工具前需要你的审批，操作透明、可撤销。
- **资料沉淀而非一次性回答**：优质回答可以保存为 Artifact，长期沉淀并可导出多格式。
- **集中凭据而非每用户 Key**（中央凭证模式）：由管理员统一发布凭证，密钥不落到客户端，用户无需关心配置；自托管模式下也可由用户自行管理 API Key。
