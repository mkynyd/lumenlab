# Skills 与 Tools

Skill 与 Tool 是 Agent 模式的两层能力描述：Tool 是 AI 可调用的单个原子能力，Skill 是把多个 Tool 按场景打包成的能力组合。

## Skill 与 Tool 的关系

一句话概括：

- Tool 是具体能做什么（例如读取项目资料、保存 Artifact）。
- Skill 是针对某个学习场景，把相关 Tool 组合起来（例如论文写作、复习辅导）。

Skill 只能从内置 Tool 中选择并收紧权限，不能引入内置 Tool 之外的操作。

## 内置 Skill 概览

LumenLab 内置了以下面向学习场景的 Skill：

| Skill | 典型用途 |
|-------|----------|
| 论文写作 | 结构化报告、引用资料原文、明确出处 |
| 复习辅导 | 复习计划、解题步骤、易错点提示 |
| 代码阅读 | 代码解释、架构分析、关键路径解读 |
| 论文速读 | 论文速读、引用整理、对比分析 |
| 考点抽取 | 严格基于资料的考题要点抽取 |
| 苏格拉底导师 | 苏格拉底式引导，拆思路不直接塞答案 |

## AI 能帮你做什么

在 Agent 模式下，AI 可能使用以下能力来协助你：

- 列出或读取当前项目的资料。
- 在项目资料中做关键词检索。
- 把当前回答保存为 Artifact。
- 导出 Artifact 为 DOCX 或 PDF。
- 搜索或抓取公开网页。
- 搜索和读取 arXiv 论文。
- 添加、列出、挂载和格式化参考文献。

> 这些能力将在后续版本中逐步开放。
