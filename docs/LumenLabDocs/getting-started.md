## 准备工作

- 一台能上网的电脑，推荐 Chrome、Edge、Safari 等现代浏览器。
- 一枚有效的 Alpha 注册码（由管理员或部署这个项目的开发者发放）。
- 建议提前准备好需要上传的课件、PDF、Office 文档、图片、代码或笔记文件。

## 注册与登录

1. 在浏览器中打开 LumenLab 网址。
2. 点击 **注册**，填写邮箱、密码和注册码。
3. 注册成功后，使用邮箱和密码 **登录**。

> 提示：注册码在服务端配置，失效或错误会导致注册失败。如提示注册码无效，请联系管理员确认。需要更换注册码时，可以调用 `POST /api/user/switch-code` 或在管理员协助下完成。

## 开始第一次对话

1. 登录后进入侧边栏的 **聊天**。
2. 在输入框上方选择模型：DeepSeek V4 Pro、Flash 或 MiniMax M3。部署者启用 Qwen 灰度且当前账号具备百炼凭据时，还会显示 Qwen3.7-Plus。
3. 在输入框工具栏点击 **Skill 选择器**，可手动切换 Skill（论文阅读、论文写作、考点提取、备考教练、代码阅读、苏格拉底式导师），或关闭 Skill 使用通用模式。
4. 输入问题，例如：
   - “帮我解释递归函数。”
   - “把这段代码改成 Python。”
   - “给一份操作系统复习提纲。”
5. 模型会以流式方式返回回答，支持 Markdown、公式、代码高亮。

> 提示：需要基于真实资料回答时，请先将资料上传到项目中，再进入项目聊天提问。

模型选择以服务端返回的目录为准。Qwen3.7-Plus 支持文本输出与图像、视频理解；图片、PDF 项目解析仍由 MiniMax M3 处理。

## 创建项目并上传资料

1. 在侧边栏点击 **项目** → **新建项目**。
2. 选择项目类型：
   - **实验**：适合实验报告、代码调试、计算过程。
   - **复习**：适合课件整理、知识点归纳、复习提纲。
   - **编程**：适合代码说明、算法分析、项目文档。
   - **通用**：适合日常笔记、自由问答。
3. 填写项目名称后进入项目工作台。
4. 在资料区点击 **上传文件**，支持 PDF、Office/WPS/iWork、图片、文本、表格和代码文件。
5. 上传后系统会自动解析内容，稍等片刻即可看到解析状态变为完成。
6. 上传限制为单文件 50MB、单次最多 50 个文件、总大小 300MB。

## 基于资料提问

1. 进入项目内的 **项目聊天**。
2. 在资料区勾选想要引用的文件，或打开「资料图谱」查看 topic / file / chunk 的关系。
3. 输入与资料相关的问题，例如：
   - “根据这份课件总结第三章重点。”
   - “结合 PDF 内容回答课后第二题。”
   - “把上传的实验数据整理成表格。”
4. AI 会结合已勾选的资料生成回答，并在回答下方显示 **深入理解**、**抓考试重点**、**生成速记卡**、**保存为成果** 等快捷操作按钮。
5. 当回答引用了项目文件、网页或 arXiv 论文时，消息底部会展示来源 UI，列出具体引用出处。

> 提示：如果勾选的文件没有进入上下文，请确认文件已完成解析，或尝试减少同时引用的文件数量。

## 保存与导出成果

1. 当 AI 的回答对你有价值时，点击回答下方的 **保存为成果**。
2. 选择成果类型，例如实验报告、复习提纲、代码说明等。
3. 保存后进入 **成果库**，可以查看、复制或导出为 Markdown、DOCX、PDF。

## 转换 PDF 文档

LumenLab 提供独立的文档转换工具，适合把整本课件、论文或扫描资料转成可编辑结构。

1. 点击顶部或侧边栏的 **工具** 入口，进入 `/tools` 页面。
2. 上传需要转换的 PDF 文件。
3. 系统会生成包含公式、表格、图片的 Markdown，并提供完整 ZIP 包下载。
4. 你也可以把转换结果直接保存为某个项目的资料。

> 提示：转换工具适合整篇文档；如果只是想让 AI 回答基于某份 PDF，直接上传到项目即可。
