## 账号与注册

**问：注册需要哪些信息？**

答：邮箱、密码和注册码。当前版本注册必须使用有效的 Alpha 注册码，没有绕过方式。

**问：注册码填错了怎么办？**

答：重新发起注册即可，注册失败不会消耗注册码。

**问：如何重置密码？**

答：当前版本暂未提供自助重置流程，需联系管理员重置。

**问：AI 服务的 API Key 在哪里配置？**

答：不需要你配置。API Key 由管理员通过注册码对应的凭证配置集中下发，用户无法查看明文。

**问：自托管部署可以让用户自己配置 API Key 吗？**

答：可以。部署者开启 `USER_API_KEYS_ENABLED=1` 后，系统会优先读取用户级 `ApiKey` 记录；未找到时再回退到中央凭证组。

## 聊天与模型

**问：为什么没有自动激活某个 Skill？**

答：Skill Router 会根据提问、选中文件、项目上下文和手动开关判断。如果任务太泛、缺少资料，可能保持通用对话。你可以在输入工具栏手动切换 Skill，或先上传/勾选相关资料。

**问：深度推理模式有什么副作用？**

答：响应时间会变长，token 用量也会增加。适合复杂推理与多步问题。

**问：对话上下文最长支持多少？**

答：由模型窗口决定，应用层目前不会主动截断历史消息。

## 项目与资料

**问：项目类型可以中途切换吗？**

答：项目高级配置功能即将上线，敬请期待。

**问：上传文件最大多少？**

答：当前限制为单文件 50MB，单次最多 50 个文件，总大小 300MB。

**问：扫描型 PDF 怎么识别？**

答：所有 PDF 均由 MiniMax M3 原生文档解析处理，不区分文本型还是扫描型，也不依赖浏览器端的 PDF 工具。

**问：PPT、Word、Excel 能直接上传吗？**

答：可以。Office、WPS 和 iWork 文档会通过 MinerU 解析为 Markdown，文档内图片会作为项目资源保存。该能力需要配置 MinerU Token。

**问：为什么我勾选的文件没有进入上下文？**

答：可能原因包括：文件尚未完成解析、上下文长度超出限制、或当前项目类型未启用该资料类型。可在资料列表查看文件状态。

## 成果与导出

**问：Artifact 支持哪些类型？**

答：支持十余种类型，例如实验报告、计算过程、复习提纲、代码说明、思维导图、通用成果等。

**问：导出格式之间可以互转吗？**

答：可以，系统内部以 Markdown 作为唯一正文源，导出为 DOCX 或 PDF 时格式保持一致。

**问：Artifact 导出的 ZIP 包含什么？**

答：Artifact 导出为单文件（Markdown / DOCX / PDF），不包含 ZIP。带图片目录的完整 ZIP 包属于文档转换工具的输出。

**问：导出失败怎么办？**

答：请检查网络连接，稍后重试。如果问题持续，请联系管理员并提供失败的成果名称与导出格式。

## Agent 模式

**问：为什么我看不到 Agent 入口？**

答：当前没有独立的 Agent 入口。Agent 事件会内联显示在对话中，由审批卡片触发确认。

**问：审批卡片一直等待，如何取消？**

答：点击 **拒绝**，或刷新页面。刷新会清空未完成的审批，需要重新发起对话。

**问：为什么 Tool 调用被拒绝了？**

答：可能原因包括：该工具未在当前 Skill 中启用、权限不足、触发了系统安全策略、风险等级超限或参数校验失败。

**问：模型会不会偷偷修改请求参数？**

答：不会。审批令牌和参数摘要会在你允许后重新校验，如果参数被替换，系统会拒绝执行该操作。
