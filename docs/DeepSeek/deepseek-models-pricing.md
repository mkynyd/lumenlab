# DeepSeek 模型与价格

## DeepSeek-V4-Pro

- **模型名**: `deepseek-v4-pro`
- **上下文长度**: 1M tokens
- **最大输出**: 384K tokens
- **思考模式**: 支持（默认启用）
- **并发限制**: 500 个请求

### 价格（每百万 tokens）

| 类型 | 价格（人民币） |
|------|---------------|
| 输入（缓存命中）| ¥0.025 |
| 输入（缓存未命中）| ¥3.00 |
| 输出 | ¥6.00 |

## DeepSeek-V4-Flash

- **模型名**: `deepseek-v4-flash`
- **上下文长度**: 1M tokens
- **最大输出**: 384K tokens
- **思考模式**: 支持（默认启用）
- **并发限制**: 2,500 个请求

### 价格（每百万 tokens）

| 类型 | 价格（人民币） |
|------|---------------|
| 输入（缓存命中）| ¥0.02 |
| 输入（缓存未命中）| ¥1.00 |
| 输出 | ¥2.00 |

## 功能支持矩阵

| 功能 | V4-Pro | V4-Flash |
|------|--------|-----------|
| JSON Output | ✅ | ✅ |
| Tool Calls | ✅ | ✅ |
| Chat Prefix Completion (Beta) | ✅ | ✅ |
| FIM Completion (Beta) | ✅ (非思考模式) | ✅ (非思考模式) |
| KV Cache | ✅ | ✅ |

## 计费说明

- Token 消耗量 × 模型单价 = 费用
- `usage` 对象中返回实际消耗 token 数
- 所有模型支持上下文缓存（KV Cache），缓存命中享受更低价格
