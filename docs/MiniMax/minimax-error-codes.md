# MiniMax API 错误码

> 来源: https://platform.minimaxi.com/docs/api-reference/errorcode

## 概述

当 API 调用出错时，请在反馈中提供 Header 中的 `trace_id`，以便排查问题。

---

## 错误码列表

### 系统与服务错误

| 错误码 | 说明 | 处理建议 |
|--------|------|---------|
| 1000 | 未知错误 | 重试 |
| 1001 | 请求超时 | 重试 |
| 1002 | 速率限制触发 | 等待后重试 |
| 1003 | 超时 | 重试 |
| 1013 | 内部服务器错误 | 联系支持 |
| 1024 | 内部故障 | 联系支持 |
| 1033 | 系统错误/下游服务错误 | 联系支持 |
| 1041 | 连接数限制 | 联系 MiniMax 支持 |

### 认证与授权

| 错误码 | 说明 | 处理建议 |
|--------|------|---------|
| 1004 | 认证失败（Token/Cookie 缺失或不匹配） | 检查凭证 |
| 2049 | 无效的 API Key | 确认 API Key 正确 |
| 2042 | 无权访问其他用户的 voice_id（仅创建者可访问） | 使用自己的 voice_id |

### 计费与配额

| 错误码 | 说明 | 处理建议 |
|--------|------|---------|
| 1008 | 余额不足 | 充值 |
| 2056 | 超出 Token Plan 资源限制 | 等待资源重置后重试 |

### 内容安全

| 错误码 | 说明 | 处理建议 |
|--------|------|---------|
| 1026 | 输入敏感内容 | 调整输入 |
| 1027 | 输出内容错误（敏感内容） | 调整输入 |

### 参数错误

| 错误码 | 说明 | 处理建议 |
|--------|------|---------|
| 2013 | 参数错误 | 检查参数 |
| 1039 | Token 限制超出 | 调整 max_tokens |
| 1042 | 不可见/非法字符超过输入的 10% | 清理输入 |

### 音色克隆相关 (20xxx)

| 错误码 | 说明 |
|--------|------|
| 20132 | 示例音频或 voice_id 参数不正确 |
| 2037 | 音频文件必须在 10 秒到 5 分钟之间 |
| 2038 | 音色克隆需完成个人或企业实名认证 |
| 2039 | voice_id 重复 |
| 2045 | 请求频率突增 |
| 2048 | prompt_audio 文件需在 8 秒以内 |

## Anthropic 兼容接口 HTTP 状态码

| HTTP 状态码 | 错误类型 | 场景 |
|-------------|---------|------|
| 400 | `invalid_request_error` | 无效参数、不支持的内容类型 |
| 401 | `authentication_error` | 缺少/无效 API key |
| 403 | `permission_error` | 无权限访问模型或路径 |
| 404 | `not_found_error` | 模型未找到 |
| 413 | `request_too_large` | 请求体 > 64 MB 或媒体超限 |
| 429 | `rate_limit_error` | RPM/TPM/并发限流 |
| 500 | `api_error` | 内部服务器错误 |
| 529 | `overloaded_error` | 上游模型过载，可重试 |
