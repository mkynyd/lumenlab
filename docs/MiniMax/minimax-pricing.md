# MiniMax Token Plan 定价

> 来源: https://platform.minimaxi.com/docs/guides/pricing-token-plan

Token Plan 订阅通过订阅 Key 提供套餐内月度用量额度，文本、图片、语音和音乐资源共享同一额度。

---

## 月订阅套餐

| 套餐 | 价格 | 适用场景 |
|------|------|---------|
| Plus | ¥49/月 | 轻量个人开发与日常试用，约 3-4 个 Agent |
| Max | ¥119/月 | 高频编程 Agent 与多模态调用，约 4-5 个 Agent |
| Ultra | ¥469/月 | 重度 Agent 工作流与更长时间使用，约 6-7 个 Agent |

所有套餐：
- 5 小时固定窗口 + 周窗口额度管理
- 覆盖开放平台所有模型

## 语言模型按量计费参考

（Token Plan 额度按此价格折算消耗）

### MiniMax-M3

| 上下文 | 等级 | 输入 (¥/1M tokens) | 输出 | 缓存读取 |
|--------|------|-------------------|------|---------|
| ≤512K | Standard | 2.10 | 8.40 | 0.42 |
| >512K | Standard | 4.20 | 16.80 | 0.84 |
| ≤512K | Priority | 3.15 | 12.60 | 0.63 |
| >512K | Priority | 6.30 | 25.20 | 1.26 |

（M3 享有永久 5 折优惠）

> Priority 通过设置 `service_tier: "priority"` 启用，花费 1.5× 标准费率。

### MiniMax-M2.7

- 输入：¥2.10 / 1M tokens
- 输出：¥8.40 / 1M tokens
- 缓存读取：¥0.42
- 缓存写入：¥2.625
- 高速版输入/输出价格翻倍

历史模型（M2.5, M2.1, M2）定价与 M2.7 相同或高速版 2×。
